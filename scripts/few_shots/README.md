### Few-Shot Evaluation Tools
This folder contains:
- `few_shot_hyper_test.py` - hyper parameter sweep
- `train_best_model.py` - final training
- `eval_final_model.py` - final evaluation
Use these to evaluate K-shot performance, run prompt-aware models, and explore mixing weights.

Outputs:
- sweep JSONs
- `final_model.pt`
- final performance JSON
